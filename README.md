Pangrams
---
file: pangrams.txt

8 pangram (pangramic?) sentences. A pangram is a sentence with every letter of the alphabet. Sounds appropriate.

Taken from: http://rinkworks.com/words/pangrams.shtml
Used the shorter ones, minus the very weird.
